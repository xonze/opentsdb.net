OpenTSDB |version| 文档
====================================

OpenTSDB |version| 是一个可扩展、分布式的时间序列数据库。 假如你需要二次开发，我们推荐你从 :doc:`user_guide/index` 的 :doc:`installation` 测试开始阅读，逐渐了解 :doc:`api_http/index`。
